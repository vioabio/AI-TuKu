1.编写用户管理页面（UserManagePage.vue）
- 搜索表单：支持按账号（userAccount）和用户名（userName）进行搜索
- 数据表格：展示用户 id、账号、用户名、头像（Image 组件）、简介、用户角色、创建时间、操作列
- 用户角色展示：管理员显示绿色 Tag，普通用户显示蓝色 Tag
- 创建时间格式化：使用 dayjs 格式化为 YYYY-MM-DD HH:mm:ss
- 分页功能：支持页码切换、每页条数切换，显示总条数
- 删除用户功能：点击删除按钮调用 deleteUserUsingPost 接口，删除后自动刷新列表
- 数据请求：通过 listUserVoByPageUsingPost 接口分页获取用户列表，页面加载时自动请求

2.全局权限控制（access.ts）
- 创建 src/access.ts 文件，在 main.ts 中引入
- 使用 Vue Router 的 beforeEach 导航守卫实现全局权限校验
- 首次加载时自动调用 fetchLoginUser 获取当前登录用户信息
- /admin 开头的路由仅允许 userRole 为 admin 的用户访问
- 无权限用户访问时提示”没有权限”，并重定向到登录页（携带 redirect 参数）
- 用户登录后可自动跳回之前访问的管理页面

3.完善全局导航栏（GlobalHeader.vue）
- 用户头像下拉菜单：登录后显示用户头像和用户名，点击展开下拉菜单
- “我的空间”菜单项：跳转到 /my_space 路由
- “退出登录”功能：调用 userLogoutUsingPost 接口，清除登录状态，跳转到登录页
- 菜单权限过滤：/admin 开头的菜单项仅对 admin 角色用户可见
  - 使用 computed 属性动态过滤菜单项
  - 非管理员用户看不到”用户管理”菜单
