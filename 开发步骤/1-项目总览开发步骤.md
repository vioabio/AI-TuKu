## 一、项目介绍 - 鱼图图

智能协同云图库（鱼图图）是一个**前后端全栈项目**，分为三大开发阶段：

| 阶段 | 主题 | 核心内容 |
|------|------|----------|
| 第一阶段 | 公共图库平台 | 用户模块、图片上传/搜索/查看、管理员管理 |
| 第二阶段 | 用户私有图库 | 私有空间、空间权限、图片编辑/AI编辑、图库分析 |
| 第三阶段 | 团队共享图库 | 团队空间、成员管理、图片协同编辑 |

### 项目三大阶段

1. **公共图库平台**：所有用户共享图片资源，管理员统一管理
2. **用户私有图库**：每个用户拥有自己的图库空间，支持个性化管理
3. **团队共享图库**：支持多人协作编辑和管理图片

## 三、核心业务流程

### 第一阶段 - 公共图库平台

```
用户注册 → 用户登录 → 浏览图片列表 → 搜索图片 → 查看图片详情 → 下载图片
                                                              ↑
管理员上传图片 ← 管理员管理图片 ← 管理员审核用户上传图片 ← 用户上传图片
```

### 第二阶段 - 用户私有图库

```
用户开通私有空间 → 用户上传图片到私有空间 → 空间级别/限额管控
                                              ↓
                                        图片编辑（基础/AI）
                                              ↓
                                        图库分析看板
```

### 第三阶段 - 团队共享图库

```
创建团队空间 → 邀请成员 → 设置权限 → 成员协作编辑图片
```

| 功能 | 说明 |

## 六、架构设计

项目采用前后端分离架构：

```
┌─────────────────────────────────────────────────┐
│                    前端 (Vue 3)                   │
│  Ant Design Vue + Pinia + Axios + Vue Router     │
└──────────────────────┬──────────────────────────┘
                       │ HTTP / WebSocket
┌──────────────────────┴──────────────────────────┐
│                  Nginx 反向代理                    │
└──────────────────────┬──────────────────────────┘
                       │
┌──────────────────────┴──────────────────────────┐
│               后端 (Spring Boot)                  │
│  Controller → Service → Mapper → DB              │
│  + Sa-Token + Redis + COS + AI                  │
└──────────────────────┬──────────────────────────┘
                       │
┌──────────────────────┴──────────────────────────┐
│           MySQL + Redis + 腾讯云 COS              │
└─────────────────────────────────────────────────┘
```

## 关键注意事项

| 注意点 | 说明 |
|--------|------|
| **首次学习建议** | 建议先学 `yu-picture-backend`（传统分层），有一定经验后再看 DDD 版本 |
| **JDK 版本** | 推荐 JDK 11 或 17，不超过 17 |
| **Node.js 版本** | >= 18.12，推荐 20 LTS |
| **前后端分离** | 部分章节的前后端可独立学习 |
